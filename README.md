# Laravel + Vue.JS Skeleton APP

INSTALL
--------------
 - Install Dependencies:
```composer install```
```npm install```
or
```yarn install```

 - Init Config:
```cp .env.example .env```
```php artisan key:generate```
 - Adjsut Config...

 - Migrate Database:
```php artisan migrate```

 - Dusk Install
```php artisan dusk:install```

 - Build Assets:
```npm run production```
